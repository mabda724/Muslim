# Muslim

تطبيق إسلامي شامل لأوقات الصلاة والأذكار، مُصمم لمساعدتك على الحفاظ على صلواتك في مواقيتها.

## ✨ الميزات

### 🕌 أوقات الصلاة
- تحديد أوقات الصلاة بدقة حسب الموقع الجغرافي
- مزامنة مع خدمة Aladhan العالمية أو الاعتماد على الحساب الفلكي المحلي
- دعم طرق حساب متعددة (مصر، كراتشي، أم القرى، رابطة العالم الإسلامي)

### 🔔 التنبيهات
- تنبيه صوتي كامل (أذان) عند دخول وقت الصلاة
- تنبيه تذكير قبل الصلاة بقابلية التخصيص
- تنبيه إقامة الصلاة بعد دخول الوقت بوقت محدد
- إشعار مستمر لمراقبة أوقات الصلاة

### 📲 حجب التطبيقات
- حجب تلقائي للتطبيقات المحددة أثناء وقت الصلاة
- طبقة حظر (Overlay) تعيد المستخدم للشاشة الرئيسية
- قائمة تطبيقات محظورة ومسموحة قابلة للتخصيص
- استثناءات للتطبيقات الأساسية (واتساب، الرسائل)

### 📊 تتبع الصلوات
- تسجيل الصلوات اليومية والأسبوعية والشهرية
- سلسلة الأيام المتتالية (Streak)
- متوسط الأسبوع وأطول سلسلة
- إحصائيات شاملة لجميع الفترات
- مؤشر تقدم بصري

### 📿 الأذكار
- أذكار يومية متجددة تلقائياً
- صلاة على النبي ﷺ
- استغفار
- نصائح إيمانية

### 🎛️ التحكم
- كتم تنبيهات مؤقت لأوقات الاجتماعات والمناسبات الحرجة
- تفعيل/تعطيل أي صلاة على حدة
- دعم التحقق البيومتري/PIN لحماية التطبيق
- إدارة مصادقة التطبيق

### 🔋 الأداء
- جلب سريع وغير متزامن لقائمة التطبيقات
- شاشة تحميل (Skeleton Loading) للتجربة السلسة
- تخزين مؤقت لأوقات الصلاة اليومية
- عمل في الخلفية بدون استهلاك بطارية

## 🛠️ التقنيات

- **اللغة:** Kotlin
- **المنصة:** Android
- **API:** Aladhan API لحساب أوقات الصلاة
- **الخدمات:** Background Service, AlarmManager, Location Services
- **التخزين:** SharedPreferences

## 🏗️ البناء

### المتطلبات
- Android Studio
- JDK 17
- Android SDK API 34

### خطوات البناء
```bash
# ضبط متغير JAVA_HOME
$env:JAVA_HOME = "C:\Program Files\Android\Android Studio\jbr"
$env:PATH = "$env:JAVA_HOME\bin;$env:PATH"

# بناء نسخة الإصدار
./gradlew assembleRelease
```

سيتم إخراج الملف في:
```
app/build/outputs/apk/release/app-release.apk
```
